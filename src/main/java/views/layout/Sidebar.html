Rules on how to play:

  • Deal four cards in a row face up.
  • If there are two or more cards of the same suit, discard all but the highest-ranked card of that suit.
  • Repeat step 2 until there are no more pairs of cards with the same suit
  • If there is/are any empty space(s), you may choose the top card or another pile to be put into the empty space.
  • When there are no more cards to move or remove, go to step 1
  • When all the cards have been dealt out and any moves made, the game is over
  • To win, the table can only contain four aces.Otherwise you lose.
